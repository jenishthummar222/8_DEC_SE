Arrays in C

    -> An array is a collection of elements of the same data type stored in contiguous memory locations. 
    -> Arrays are used to store multiple values in a single variable and are accessed using an index.

    
    Syntax for Declaration:

        data_type array_name[size];

    Example:

        int marks[5]; // array of 5 integers
    
1. One-Dimensional (1D) Arrays

    -> A 1D array is a single row of elements.

    -> Accessed using one index.

    -> Useful for storing a list of values.

    Example:

        #include <stdio.h>
        int main() 
        {
            int marks[5] = {80, 85, 90, 75, 95};
            for(int i = 0; i < 5; i++) 
            {
                printf("%d ", marks[i]);
            }
            return 0;
        }

    OutPut:-
        80 85 90 75 95

2. Multi-Dimensional Arrays

    -> A multi-dimensional array is an array of arrays.

    -> Most commonly, a 2D array is used (rows Ã— columns).

    -> Accessed using two or more indices.

    -> Useful for tables, matrices, or grids.

    Example (2D Array):

        #include <stdio.h>
        int main() 
        {
            int matrix[2][3] = { {1, 2, 3}, {4, 5, 6} };
            for(int i = 0; i < 2; i++) 
            {
                for(int j = 0; j < 3; j++) 
                {
                    printf("%d ", matrix[i][j]);
                }
                printf("\n");
            }
            return 0;
        }


    Output:

        1 2 3
